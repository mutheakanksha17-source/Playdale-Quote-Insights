# Playdale Quote Insights

A machine learning and analytics project focused on understanding the factors influencing quote-to-order conversion in B2B sales environments.

This project was completed as part of an industry dissertation placement with Playdale Playgrounds Ltd.

---

## Business Problem

In B2B sales environments, companies generate large volumes of quotations, but not all convert into successful orders.

This project investigates:
- which factors influence quote success
- how client and regional patterns affect conversion
- how machine learning can support quotation decisions

---

## Objectives

- Clean and standardise sales quotation data
- Prevent data leakage in predictive modelling
- Identify key drivers of quote conversion
- Compare interpretable and ensemble ML models
- Analyse product success patterns across regions and client types

---

## Models Used

- Logistic Regression
- Random Forest
- Gradient Boosting

---

## Technologies Used

- Python
- pandas
- scikit-learn
- NumPy
- Matplotlib
- Jupyter Notebook

---

## Key Findings

- Geography and client type were the strongest predictors of quote conversion.
- Random Forest achieved the best predictive performance.
- Seasonal patterns influenced conversion behaviour.
- Product success patterns varied across regions and client segments.
